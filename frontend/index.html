🚀 Как использовать YouTube Analyzer

  1. 1
    Получите YouTube API ключ

    Перейдите в Google Cloud Console и создайте API ключ для YouTube Data API v3

  2. 2
    Настройте приложение

    Добавьте API ключ в настройках и установите лимиты обработки

  3. 3
    Создайте задачу анализа

    Укажите ключевые слова или URL каналов для анализа

  4. 4
    Изучите результаты

    Просматривайте видео и каналы, экспортируйте данные в Excel

❓ Частые вопросы

Сколько видео можно анализировать?

YouTube API имеет лимит 10,000 запросов в день. Рекомендуем анализировать не более 50-100 видео за раз.

Как работает транскрибация?

Сначала пытаемся получить субтитры YouTube. Если их нет, используем Whisper AI для транскрибации аудио.

Можно ли поставить задачу на паузу?

Да, любую активную задачу можно поставить на паузу и возобновить позже с того же места.
